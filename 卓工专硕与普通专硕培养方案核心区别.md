## 卓工专硕与普通专硕培养方案核心区别

#### Q:普通专硕拿到学位证需要什么条件？

1.修读完全部学分（课程，专业实践（实习），文献综述，开题报告）

2.核心期刊发表学术论文或申请国家发明专利（俗称“小论文”）

3.完成学位论文并通过答辩（俗称“大论文”）

#### Q:卓工专硕和普通专硕在拿到学位证的条件上有什么区别？

一般而言第一项与普通专硕一致；第三项也与普通专硕一致，不过学位论文的选题要来源于工程实际，可选用实践成果答辩替代；最大的区别在于第二项，**国家政策明确规定不以发表学术论文作为研究生毕业及申请学位的前置条件**，而用 **工程新技术/重大工程设计/新产品新装置研制** 等成果取而代之 

其次，普通专硕规定专业实践（实习）要大于6个月，而卓工专硕已经提前确定实习企业，并且在企实习时间要大于12个月



**允许的成果形式：**

| 形式               | 说明                               |
| ------------------ | ---------------------------------- |
| 工程新技术研究     | 以文字形式系统表述工程技术研究成果 |
| 重大工程设计       | 面向实际工程项目的设计方案和论证   |
| 新产品或新装置研制 | 产品研发全过程的技术文档           |
