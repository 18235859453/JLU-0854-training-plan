# 国家发明专利申请指南

> **适用对象**: 吉林大学电子科学与工程学院专业学位硕士研究生  
> **毕业要求**: 申请国家发明专利 1 项（以受理时间为准）  
> **核心认知**: **只需拿到受理通知书即满足毕业条件，不需要等到授权。**

---

## 一、整体流程一览

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 准备材料  │ → │ 校内审批  │ → │ 提交申请  │ → │ 形式审查  │ → │ 初步审查  │ → │ 实质审查  │ → 授权
│ 1-3 周    │   │ 1 周     │   │ 1 天      │   │ 1-2 周   │   │ 3-6 个月  │   │ 1-3 年    │
│           │   │          │   │           │   │ 🎯 受理    │   │           │   │           │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
                                                     ↑
                                                 拿到受理通知书 = 满足毕业条件 ✅
```

---

## 二、分步骤详解

### 第 1 步：准备技术交底书（1-3 周）

**与导师充分沟通后**，准备一份技术交底书，回答以下问题：

| 要素 | 内容 |
|------|------|
| **要解决的问题** | 现有技术存在什么缺陷、痛点？ |
| **技术方案** | 你的发明是什么？（原理、结构、流程、算法等） |
| **与现有技术的区别** | 和已有方案比，你的创新点在哪？ |
| **具体实施方式** | 至少给出 1-2 个具体实现例子 |
| **附图** | 结构图、流程图、原理框图等 |

> 💡 技术交底书可以基于你学位论文的阶段性成果来写，两者不冲突。

**⚠️ 保密提醒**: 在提交专利申请之前，**不要**公开发表论文、不要在学术会议上报告、不要在社交平台或任何公开场合披露技术细节。一旦公开，你的发明就丧失了"新颖性"，专利将无法授权。

---

### 第 2 步：专利检索（建议做）

在提交前，登录[国家知识产权局专利检索系统](https://pss-system.cponline.cnipa.gov.cn)检索是否已有相同或相似的专利。

- 如果检索到雷同专利 → 调整技术方案，强调不同点
- 如果没有雷同 → 放心推进

> 代理机构提交时也会做检索，但自己先做一遍更放心。

---

### 第 3 步：校内审批（约 1 周）

吉林大学要求职务发明（即你在校期间利用学校资源完成的发明）必须通过校内审批：

1. 填写 **《吉林大学专利申请审查表》**
2. **导师签字**
3. 到**学院加盖公章**

审查表向学院研究生办公室索取，或从学校科技处网站下载。

---

### 第 4 步：选择申请方式

#### 方式一：委托代理机构（推荐 ✅）

| 项目 | 说明 |
|------|------|
| **你做什么** | 提供技术交底书，与代理人沟通技术细节 |
| **代理做什么** | 撰写全套申请文件（请求书、说明书、权利要求书、摘要），提交申请，答复审查意见 |
| **费用** | 代理费 5000-15000 元/件 |
| **时间** | 交底后 1-3 周完成文件撰写并提交 |

**吉林大学长期合作的代理机构**：

| 机构 | 说明 |
|------|------|
| **长春吉大专利代理有限责任公司** | 前身吉林大学专利事务所（1985年成立），已代理吉大专利超 11,000 件，首选推荐 |

> 导师通常有合作的代理机构，先问导师的建议。

#### 方式二：自行申请

| 项目 | 说明 |
|------|------|
| **你做什么** | 自己撰写全套申请文件，在[专利业务办理系统](https://cponline.cnipa.gov.cn)注册并提交 |
| **费用** | 仅官费（减缴后约 583 元） |
| **适合情况** | 技术方案简单、有人指导、不想花代理费 |

自行申请需要准备的文件：

| 文件 | 要求 |
|------|------|
| 发明专利请求书 | 含申请人（吉林大学）、发明人信息、发明名称 |
| 说明书 | 技术领域、背景技术、发明内容、附图说明、具体实施方式 |
| 权利要求书 | 独立权利要求 + 从属权利要求，定义保护范围（最核心文件） |
| 说明书附图 | 结构图、流程图等 |
| 说明书摘要 | 300 字以内技术概述 |
| 费用减缴请求书 | 申请减缴 85% 费用 |

---

### 第 5 步：提交申请并拿到受理通知书 🎯

无论哪种方式，提交后 **1-2 周**内国家知识产权局会发出：

> **《专利申请受理通知书》**

上面载明你的**申请号**和**申请日**（即你提交的那一天）。

**至此，你的毕业成果条件即满足。** ✅  
（授权可以慢慢等，毕业后也能继续走流程）

---

## 三、后续流程（受理之后，不影响毕业）

| 阶段 | 时间 | 说明 |
|------|------|------|
| 初步审查 | 3-6 个月 | 形式审查，文件格式是否合规 |
| 公布 | 申请日起 18 个月 | 可请求提前公布（最快 6 个月） |
| 实质审查请求 | 申请日起 3 年内必须提出 | 缴纳实审费 2500 元（减缴后 375 元） |
| 实质审查 | 提出请求后 1-3 年 | 审查新颖性、创造性、实用性 |
| 答复审查意见 | 收到通知后 4 个月内 | 修改权利要求书或陈述意见 |
| 授权 | 审查通过后 | 缴纳授权登记费，拿到专利证书 |

---

## 四、费用明细

### 4.1 官费（国家知识产权局收取）

| 费用项 | 标准金额 | 学生减缴 85% 后 |
|--------|----------|-----------------|
| 申请费 | 900 元 | **135 元** |
| 公布印刷费 | 50 元 | 50 元 |
| 实质审查费 | 2500 元 | **375 元** |
| 授权登记费 + 印花税 | ~255 元 | ~38 元 |
| 年费（第 1-3 年，每年） | 900 元 | **135 元** |
| **拿到受理通知书的官费** | **~950 元** | **~185 元** |

### 4.2 代理费（非强制）

| 类型 | 费用 |
|------|------|
| 普通代理机构 | 5000-15000 元/件 |
| 高端代理机构 | 20000-50000 元/件 |

### 4.3 费用减缴申请

作为无固定收入的研究生，符合**个人费用减缴条件**（上年度月均收入低于 5000 元）。

操作：在提交专利申请之前，先在国家知识产权局"**专利费减备案系统**"办理备案，通过后再提交专利申请即可享受 85% 减免。

## 五、署名要求

根据培养方案规定：

| 要求 | 说明 |
|------|------|
| **第一署名单位** | 吉林大学电子科学与工程学院和集成光电子全国重点实验室 |
| **发明人署名** | 第一发明人，或除本人指导教师之外的第一发明人 |
| **共同发明人** | 如有 N 个共同第一发明人，按 1/N 计算（与论文规则相同） |

> 提交前务必核对单位全称，写错可能导致成果不被认可。

## 六、替代方案

培养方案规定，以下方式**二选一**即可满足毕业要求：

| 选项 | 要求 |
|------|------|
| 国家发明专利（受理） | 1 项，以受理时间为准 |
| 核心及以上学术论文（发表或 online） | 1 篇，可替代专利 1 项 |

---

## 七、时间规划建议

```
第 1 学期    ──── 上课为主，跟导师确定大致研究方向
第 2 学期    ──── 论文工作启动，开始积累可申请专利的技术成果
第 3 学期    ──── 开题报告 + 准备技术交底书
第 3 学期末   ─── 提交专利申请 🎯 拿到受理通知书
  （或第 4 学期初）
第 5-6 学期  ──── 论文撰写、答辩，专利可继续走后续流程
```

> ⚠️ 不要拖到第 5-6 学期才提交。受理通知书拿到后还需要时间走校内成果备案流程，预留至少 2 个月的缓冲期。

## 八、常见问题

**Q1: 专利必须是授权的才算数吗？**  
不。培养方案明确写的是"申请"，以**受理时间**为准，拿到受理通知书即可。

**Q2: 专利和论文可以来自同一个研究工作吗？**  
可以。论文和专利保护的是不同东西——论文公开知识，专利保护技术方案。事实上，很多导师会建议先提交专利申请，再发表论文，这样两不耽误。

**Q3: 如果专利最终被驳回，影响毕业吗？**  
不影响。培养方案只要求"申请"，不要求"授权"。拿到受理通知书就已经满足毕业条件了。

**Q4: 专利费用谁出？**  
通常由**导师的科研经费**承担。这是常态，提交前跟导师确认费用出处。

**Q5: 实用新型专利可以吗？**  
不行。培养方案明确要求的是"**国家发明专利**"。实用新型不能替代。
