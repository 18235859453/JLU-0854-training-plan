## 三、普通专硕：拿到学位证与毕业证的完整流程

### 3.1 流程总图

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 修满学分  │ → │ 文献综述  │ → │ 开题报告  │ → │ 专业实践  │ → │ 中期考核  │ → │ 论文答辩  │
│ ≥32 学分  │   │ 第2-3学期 │   │ 第3学期末 │   │ ≥6个月    │   │ 第5学期末 │   │ 第6学期   │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
                                                                                      ↓
                                                                             ┌──────────────┐
                                                                             │ 🎓 双证到手   │
                                                                             │ 毕业证+学位证 │
                                                                             └──────────────┘
```

### 3.2 各环节详解

#### 第一步：修满培养方案规定的全部学分（≥ 32 学分）

| 学期 | 做什么 | 关键提醒 |
|------|--------|----------|
| 第1学期 | 全力修课（约 18 学分），公共课 + 专业必修 + 选修 | ⚠️ 第1学期仅有「现代数字信号处理」1门专业必修大课开设，必须选上 |
| 第2学期 | 修完剩余课程（约 8 学分），完成文献综述 | 专业必修选课累计 ≥ 9 学分即可，不贪多 |
| 第3学期 | 科学道德与学术规范，开题报告 | 学期末完成开题 |

#### 第二步：完成全部必修环节

| 环节 | 时间 | 要求 |
|------|------|------|
| 科学道德与学术规范 | 第3学期 | 考试通过 |
| 学术活动 | 全程 | ≥ 3 次学术报告总结 |
| 文献综述 | 第2-3学期 | 选题后撰写，上传系统审核 |
| 开题报告 | 第3学期末 | 学院统一组织 |
| 社会实践 | 答辩前 | 填表 → 导师签字 → 交学院备案 |
| 中期考核 | 第5学期末 | 学院统一组织 |

#### 第三步：完成专业实践（4 学分）

- 累计 **≥ 6 个月**，原则上在中期考核前完成
- 具有 2 年及以上企业工作经历者可申请免修
- 完成后提交实践报告，考核合格获得学分

#### 第四步：满足学位授予前置条件（🔴 关键门槛）

> **以下二选一，必须在答辩前完成：**

| 路径 | 要求 |
|------|------|
| 路径 A | 申请 **国家发明专利 1 项**（以受理时间为准） |
| 路径 B | 发表 **核心及以上学术论文 1 篇**（含 online） |

**署名要求**：
- 第一作者（或除本人指导教师之外的第一作者）
- 第一署名单位：**吉林大学电子科学与工程学院和集成光电子全国重点实验室**

#### 第五步：完成学位论文并通过答辩

- 在导师指导下独立完成学位论文
- 论文评阅：5 名高级职称专家
- 答辩委员会：5 名高级职称专家（导师不得任主席）
- 答辩通过 → 🎓 授予 **工程硕士学位** + **毕业证书**

## 四、卓越工程师学院（珠海）专硕：拿到学位证与毕业证的流程

### 4.1 背景

吉林大学卓越工程师学院珠海分院采用 **工程硕士培养改革专项试点** 方案，与中国一汽等企业深度联合培养，实行 **"1+2"工学交替模式**。

> 详细调查见 → [卓越工程师学院培养模式调查报告.md](卓越工程师学院培养模式调查报告.md)  
> 对比分析见 → [培养方案对比/两种培养方案对比分析.md](培养方案对比/两种培养方案对比分析.md)

### 4.2 流程总图

```
┌──────────┐    ┌──────────┐    ┌──────────────────────────┐    ┌──────────────┐
│ 第1年    │ → │ 综合考核  │ → │ 第2-3年 企业实践 + 论文    │ → │ 实践成果答辩  │
│ 在校修课  │   │ 分流筛选  │   │ 工学交替，企业真实攻关任务  │   │ 🎓 双证到手   │
└──────────┘    └──────────┘    └──────────────────────────┘    └──────────────┘
```

### 4.3 各环节详解

#### 第一步：第1年在校完成课程学习（与普通专硕一致）

学分要求与普通专硕完全相同（≥ 32 学分），课程表也基本一致。部分专业课由**企业技术专家授课**，鼓励将课堂移至企业。

#### 第二步：综合考核与分流

第1年末由校企双方进行综合考核。**不符合培养要求的学生将被分流**至普通专硕通道。

#### 第三步：第2-3年在企业实践（🟢 最大区别）

| 维度 | 普通专硕 | 卓工专硕 |
|------|----------|----------|
| 实践时长 | ≥ 6 个月 | **≥ 12 个月**（实际约 2 年） |
| 实践地点 | 校内/校外灵活 | 企业（中国一汽） |
| 实践内容 | 自行安排或导师安排 | **企业真实研发攻关任务** |
| 导师制度 | 校内导师为主 | **校企导师组**（校内导师 + 企业导师，双方各设责任导师） |

校企合作三种模式：
1. **依托在研合作项目** — 校企已联合立项的重大工程项目
2. **"揭榜挂帅"** — 企业发布技术难题 → 学校导师揭榜 → 联合培养
3. **依托企业自研项目** — 企业导师根据自研需要提出任务计划

#### 第四步：学位成果 — 🔴 最大优势

> **国家政策明确规定：不以发表学术论文作为研究生毕业及申请学位的前置条件。**

不需要申请专利，也不需要发表核心期刊论文。取而代之的是以下实践成果：

| 成果形式 | 说明 |
|----------|------|
| 工程新技术研究 | 以文字形式系统表述工程技术研究成果 |
| 重大工程设计 | 面向实际工程项目的设计方案和论证 |
| 新产品或新装置研制 | 产品研发全过程的技术文档 |
| 专利 / 软著 | 实践过程中产生的知识产权 |
| 标准制定 / 获奖成果 | 其他形式的创新成果 |

法律依据：《中华人民共和国学位法》（2025.1.1 施行）第二十条明确允许专业学位申请人通过**"实践成果答辩"**申请学位。

#### 第五步：实践成果答辩

| 维度 | 普通专硕 | 卓工专硕 |
|------|----------|----------|
| 论文选题 | 导师指导决定 | **必须来源于工程实际**，与专业实践紧密联系 |
| 成果形式 | 学术型学位论文 | 工程新技术/重大工程设计/新产品新装置研制 |
| 评阅专家 | 5 名校内专家 | 3 名专家，**企业专家占半数以上** |
| 答辩委员会 | 5 名校内专家 | 5 名专家，**企业专家占半数以上**，校企联合组织 |

## 五、两种路径核心对比

| 对比维度 | 普通专硕 | 卓工专硕（珠海） |
|----------|----------|-----------------|
| **培养地** | 3 年在校 | 1 年在校 + 2 年在企业 |
| **导师** | 校内导师为主 | 校企导师组（双责任导师） |
| **专业实践** | ≥ 6 个月，可申请免修 | ≥ 12 个月，不可免修，全程企业负责 |
| **学位前置条件** | 🔴 专利 或 核心论文 | 🟢 **无需专利/论文** |
| **答辩方式** | 传统论文答辩 | 实践成果答辩（可替代论文） |
| **评阅/答辩专家** | 校内为主 | 企业专家占半数以上 |
| **分流机制** | 无 | 有（不符合要求则转入普通通道） |
| **学位证书** | 工程硕士学位 | 工程硕士学位（相同） |
| **毕业证书** | 毕业证书 | 毕业证书（相同） |
| **适合人群** | 大部分普通录取专硕 | 选拔进入卓工学院试点的专硕 |

### 结论

- 无论哪种路径，拿到的**学位证和毕业证完全一样**
- 卓工路径最大的优势是**免除专利/论文硬性要求**，代价是**2年绑定在企业**且有分流风险
- 普通专硕的核心挑战是**答辩前搞定专利或核心论文**
