# 核心及以上学术论文 — 写作与发表指南

> **适用对象**: 吉林大学电子科学与工程学院专业学位硕士研究生  
> **毕业要求**: 核心及以上学术论文 1 篇（以发表时间/online 时间为准，可替代国家发明专利）  
> **最低门槛**: 北大核心 或 CSCD 收录期刊

---

## 一、什么是"核心及以上"

### 1.1 层级体系

"核心及以上"是一个由低到高的层级概念：

```
中国科技核心  <  北大核心  <  CSCD / CSSCI  <  EI  <  SCI
（门槛最低）      （最通用）      （国内顶尖）           （国际）
```

### 1.2 各核心级别说明

| 级别 | 全称 | 评价机构 | 覆盖学科 | 特点 |
|------|------|----------|----------|------|
| **中国科技核心** | 中国科技论文统计源期刊 | 中国科学技术信息研究所 | 自然科学 | 每年更新，门槛最低，部分学校不认 |
| **北大核心** | 中文核心期刊要目总览 | 北京大学图书馆 | 全学科 | 国内最通用，每 3 年更新 |
| **CSCD** | 中国科学引文数据库 | 中科院文献情报中心 | 自然科学 | 自科领域国内顶尖，被誉为"中国的 SCI" |
| **CSSCI** | 中文社会科学引文索引 | 南京大学 | 人文社科 | 社科领域国内顶尖（你的专业不适用） |
| **EI** | 工程索引 | Elsevier | 工程技术 | 国际检索，认可度高 |
| **SCI** | 科学引文索引 | Clarivate | 自然科学 | 国际顶级 |

> 对你的专业（电子信息/电子科学）来说，培养方案中"核心及以上"指的是**北大核心**、**CSCD**、**EI**、**SCI** 这几个级别。  
> ⚠️ 中国科技核心（统计源核心）通常不被吉大认可为"核心"，投之前务必跟导师确认。

### 1.3 如何查询某期刊是否为核心

1. 登录[知网（CNKI）](https://www.cnki.net)，搜索期刊名称
2. 在期刊详情页查看"收录情况"
3. 或在[北大核心目录](https://lib.pku.edu.cn)、[CSCD 官网](http://sciencechina.cn)核对最新版目录

## 三、论文发表全流程

### 3.1 流程总览

```
┌──────┐   ┌──────┐   ┌──────┐   ┌──────────┐   ┌──────────┐   ┌──────┐   ┌──────┐
│ 选题  │ → │ 实验  │ → │ 写作  │ → │ 投稿初审  │ → │ 同行外审  │ → │ 修改  │ → │ 录用  │
│2-4周  │   │2-6月  │   │4-8周  │   │ 1-3周     │   │ 1-4月     │   │2-4周  │   │      │
└──────┘   └──────┘   └──────┘   └──────────┘   └──────────┘   └──────┘   └──────┘
                                                                              ↓
                                                                    网络首发（online）
                                                                    1-6 个月后见刊
```

**从开始写作到拿到录用通知，全程大约 4-10 个月。** 拿到 online 或正式录用通知即可用于毕业认定。

### 3.2 先申请专利还是先发论文？

**必须先申请专利，再发表论文。**

原因：论文一旦发表（包括 online），你的技术方案就变成了"现有技术"（公开知识），**丧失新颖性**，之后再也无法申请专利。正确顺序是：

```
先提交专利申请（拿到受理通知书）
    ↓
再投稿发表论文
    ↓
两者不冲突 ✅
```

## 四、各阶段详解

### 阶段 1：选题（2-4 周）

论文选题通常来源于：
- 导师科研项目中的子问题
- 学位论文工作的阶段性成果
- 实验中发现的有价值的问题

**选题三要素**：
1. **有创新**：和已有方法有明确区别（哪怕是小改进）
2. **有数据**：有足够的实验/仿真结果支撑
3. **可完成**：能在 2-6 个月内拿出完整结果

### 阶段 2：实验与研究（2-6 月）

- 设计实验方案
- 收集/生成数据
- 与现有方法做对比实验
- 记录所有实验参数和结果（方便后续写论文时回溯）

### 阶段 3：论文写作（4-8 周）

#### 3.1 标准结构（IMRaD）

| 部分 | 作用 | 建议篇幅 |
|------|------|----------|
| **标题** | 概括研究内容，包含关键词，20 字以内 | 1 行 |
| **摘要** | 问题 → 方法 → 结果 → 结论，200-300 字 | 1 段 |
| **关键词** | 3-8 个，便于检索 | 1 行 |
| **引言** | 背景、现有方法不足、你的贡献点 | 1-2 页 |
| **相关工作** | 已有研究的分类评述（先分类 → 再评述 → 最后指出不足） | 1-2 页 |
| **方法** | 你的方案、模型架构、算法流程 | 2-4 页 |
| **实验与结果** | 数据说明、评价指标、对比表格、可视化图表 | 2-4 页 |
| **分析与讨论** | 为什么你的方法更好？局限性？ | 1 页 |
| **结论** | 总结贡献 + 未来工作展望 | 0.5-1 页 |
| **参考文献** | 通常 20-40 篇，近 5 年文献不低于 50% | — |

#### 3.2 写作顺序建议

```
先写主体：方法 → 实验 → 相关工作
再写首尾：引言 → 结论 → 摘要
最后打磨：标题 + 反复修改
```

#### 3.3 写作工具推荐

| 工具 | 用途 |
|------|------|
| LaTeX（Overleaf/本地） | 排版（大多数核心期刊提供 LaTeX 模板） |
| Word | 排版（部分期刊仅接受 Word） |
| Zotero / EndNote | 参考文献管理 |
| Grammarly | 英文论文语法检查 |
| 知网个人查重 | 投稿前自查重复率 |

#### 3.4 查重要求

- 核心期刊查重率通常要求 **≤ 10%**（部分要求 ≤ 5%）
- 投稿前建议自己在知网查一遍，避免因查重被直接退稿
- AI 辅助写作：部分期刊要求 AI 生成内容占比不超过 15%。AI 只能用作辅助（翻译、润色），不能替代核心写作

### 阶段 4：选刊投稿

#### 4.1 选刊步骤

1. 在知网搜索你的关键词，看类似论文发在哪些期刊
2. 选中几本北大核心或 CSCD 期刊
3. 去期刊官网查看"投稿指南"（格式、字数、查重标准）
4. 按模板格式排版

#### 4.2 投稿信（Cover Letter）模板

```
尊敬的编辑老师：

   您好！

   我们投稿的论文题为《XXXXXX》，研究了[一句话概括研究内容]。

   本文的主要创新点为：
   1. [创新点一]
   2. [创新点二]
   3. [创新点三]（可选）

   本文未在其他期刊投稿或发表，所有作者均同意投稿，无利益冲突。

   感谢您百忙之中审阅！

   此致
   敬礼

   通讯作者：XXX
   日期：202X年X月X日
```

#### 4.3 投稿策略

- **从高往低投**：先投好一点的期刊，被拒后再降级
- **一次只投一本**：绝不可一稿多投（学术不端）
- **准备 2-3 本备选期刊**：被拒后能快速转投
- **有基金项目要写上**：国家自然科学基金等项目支持显著提高录用概率

### 阶段 5：审稿流程

```
投稿
  ↓
编辑初审（1-3 周）
  ├── 格式不符 → 退回修改
  ├── 方向不符 → 直接退稿
  └── 通过 ↓
送同行外审（2-4 位审稿专家，1-4 个月）
  ├── 直接录用（极其罕见）
  ├── 小修（Minor Revision）→ 修改后编辑直接决定
  ├── 大修（Major Revision）→ 修改后送回审稿人再审
  └── 退稿（Reject）→ 根据意见修改后转投其他期刊
```

### 阶段 6：回复审稿意见

这是论文能否被录用的**最关键环节**。

#### 回复格式

```
审稿人 #1 意见 1：
[原文照抄审稿意见]

回答：
感谢审稿人的宝贵意见。[解释你做了什么修改 / 为什么不同意]
我们已在文中第 X 页第 Y 段进行了相应修改（标红标注）。
```

#### 回复原则

| 原则 | 说明 |
|------|------|
| **尊重** | 每条意见先感谢，哪怕是批评 |
| **逐条回答** | 一个不漏，每条都有回应 |
| **具体** | 说明改了什么、改在哪一页 |
| **有证据** | 补充实验数据、引用文献支撑 |
| **标注清晰** | 修改处用红色/高亮标注，方便审稿人对照 |

### 阶段 7：录用 → 见刊

收到录用通知后：
1. 签署版权转让协议
2. 缴纳版面费（核心期刊通常 2000-8000 元，由导师科研经费支出）
3. 等待排版校对（期刊编辑部会发排版稿给你确认）
4. **网络首发（online）**：此时即可用于毕业条件认定
5. 纸质见刊：1-6 个月后

## 六、时间规划建议

```
第 1 学期    ————— 上课为主，跟导师确定研究方向和大致选题
第 2 学期    ————— 实验起步 + 文献积累 + 开始论文初稿
第 3 学期    ————— 完成实验 + 论文撰写 + 投出论文
第 4 学期    ————— 审稿周期中（回复修改意见、等待录用）
第 5 学期    ————— 确保拿到录用通知或 online（用于毕业认定）
第 6 学期    ————— 答辩，论文正式见刊
```

> ⚠️ 考虑到审稿可能需要多轮修改甚至被拒后转投，**最迟第 4 学期初必须投出**，否则可能来不及。

## 七、论文 vs 专利 — 对比

| 维度 | 核心论文 | 国家发明专利 |
|------|----------|-------------|
| **门槛** | 需要完整研究 + 写作 + 通过同行审稿 | 有技术方案即可申请 |
| **时间成本** | 4-10 个月（到录用） | 1-2 周（拿到受理通知） |
| **不确定性** | 可能被拒稿多次 | 受理基本确定（不审新颖性） |
| **费用** | 版面费 2000-8000 元（导师经费出） | 官费约 185 元（减缴后） |
| **学术价值** | 锻炼写作能力、学术积累 | 纯为毕业条件 |
| **读博加分** | 高 | 低 |

### 策略建议

- **以专利保底**：第 3 学期末提交专利申请，1-2 周拿到受理通知，毕业底线守住
- **论文作为进阶**：有余力就冲击核心论文，提升学术能力，对读博或找研发岗有利
- **两条腿走路最稳**：先专利后论文，顺序不能反

## 八、常见问题

**Q1: 论文可以和学位论文用同一项研究吗？**  
可以。实际上大部分研究生都是从学位论文工作中拆分出一部分来发小论文。但不能大段照抄，需要重新组织。

**Q2: 论文必须见刊才算数吗？**  
不需要。培养方案写的是"以发表时间（含 online 时间）为准"，网络首发（online）即可。

**Q3: 被退稿了怎么办？**  
很正常。根据审稿意见修改后转投下一本备选期刊。顶刊的录用率通常不到 20%，被拒是常态。

**Q4: 什么是 EI 会议论文？算核心及以上吗？**  
EI 收录的国际会议论文通常算"及以上"级别，认可度高于核心期刊。但需要注意：不是所有会议都算，必须是 EI 收录的。跟导师确认。

**Q5: 版面费谁出？**  
通常由**导师的科研经费**承担。投稿前跟导师确认。
